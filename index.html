Jquery-stalker

jQuery transition effect plugin

View the Project on GitHub gcko/jquery-stalker

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Stalker is a jQuery transition effect - when the screen is scrolled vertically, it attaches the specified element to the top of the screen. When you scroll back the element reverts to its original position. See demo.html for an example.

Usage

  jQuery("#stalker").stalker();
